# Switch Dark/Light Mode for websites.

Dark/Light Mode
508 Dark Mode is a lightweight WordPress plugin that adds a smooth, user-controlled dark mode to any website using the powerful DarkReader engine. Unlike heavy theme-based solutions, this plugin works independently from your design system and applies dark mode dynamically on the client side — without modifying your theme styles.

✨ Features
Adds dark mode support to any WordPress site
Uses DarkReader for high-quality rendering
Remembers user preference via localStorage
Works with any theme or page builder
Includes a ready-to-use shortcode [508_dark_mode]
Optional automatic injection into mobile menu
Clean toggle UI with animated switch
Loads DarkReader early to prevent flash of light theme
